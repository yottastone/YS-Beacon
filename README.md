# YS-Beacon：一款能够帮助开发者更加智能地感知世界的硬件
YS-Beacon 是一款基于Nordic nRF52832，集成了3轴加速度计、3轴陀螺仪、3轴罗盘、环境光传感器、接近传感器、3D手势识别传感器、数字大气压传感器、立体声麦克风等传感器的开源 Sensor Beacon 硬件。
